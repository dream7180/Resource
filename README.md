# foobox 格式图标 / 电台封面
1. 为 foobox 定制的音乐格式图标，foobox 7.17 版开始已经集成，不使用 foobox 也可替换原版 icons，替换后需要更新一下系统的图标缓存。

2. foobox 7.10 版开始集成了高品质的网络电台列表，这里收集了多数电台的 logo 封面，请下载（整个仓库）后放在你想放置的地方，
  在 foobar2000 参数选项的封面显示里添加两行
   ```
    [logo文件夹路径\%title%.jpg]
    [logo文件夹路径\%title%.png]
   ```
